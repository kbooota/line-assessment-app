LINE アンケート API

概要

このAPIはLINE LIFFアプリからのアンケートデータを受け取り、Googleスプレッドシートに保存します。

使用方法

このAPIはPOSTリクエストで以下のJSONデータを受け付けます：

{
  "lineUserId": "U1234567890abcdef1234567890abcdef",
  "name": "山田太郎",
  "age": "30代",
  "assessmentType": "不動産",
  "detail": "2LDKマンションの査定をお願いします"
}

レスポンス形式

成功時のレスポンス：

{
  "success": true,
  "message": "データが正常に保存されました",
  "timestamp": "2025-03-21T12:34:56.789Z"
}

エラー時のレスポンス：

{
  "success": false,
  "message": "データの保存中にエラーが発生しました",
  "error": "エラーの詳細情報"
}

セキュリティ

このAPIは公開されていますが、LINE LIFF環境からの正規のリクエストのみを想定しています。
